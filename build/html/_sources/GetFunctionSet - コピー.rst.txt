Get函数族
====================

.. toctree::
   :maxdepth: 4

函数族指的是一类函数的集合，在windget中，使用最多的是get函数族，他们是一类以get开头的函数，后面紧接的英文名称表示您想从万得获取的字段，最后还可能有函数尾缀。

最简单的例子是获取某个交易日的收益率：
::

   df = getPctChg("000001.SZ","tradeDate=20220511")

或者获取某个交易日多只股票的收益率：
::

   df = getPctChg(["000001.SZ","000004.SZ"],"tradeDate=20220511")

或者获取某天到某天的收益率序列：
::

   df = getPctChgSeries("000001.SZ","20220511","20220611")

或者获取多个股票的收益率序列：
::

   df = getPctChgSeries(["000001.SZ","000004.SZ"],"20220511","20220611")
